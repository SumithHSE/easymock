If you have only 1 minute

Add EasyMock and its dependencies to your POM file.

{% highlight xml %} org.easymock easymock 3.2 test {% endhighlight %}

In your test case, create a Mock Object for the Interface / Class to be simulated, and replace the collaborator of class under test with the mock.

{% highlight java %} import static org.easymock.EasyMock.*; import org.junit.*; public class ExampleTest { private ClassUnderTest classUnderTest; private Collaborator mock; @Before public void setUp() { mock = createMock(Collaborator.class); classUnderTest = new ClassUnderTest(); classUnderTest.setCollaborator(mock); } } {% endhighlight %}
